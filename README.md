# Valentine Vibes

A playful Valentine's Day web experience with an escaping "No" button, floating hearts, and a dreamy aurora backdrop. Built as a vibe‑coding project with a big thank‑you to Lovable AI for the creative boost and tooling magic.

## Features
- Valentine setup flow with shareable link generation
- Escaping "No" button interactions with playful text changes
- Aurora background + floating hearts ambience
- Responsive layout for mobile and desktop

## Tech Stack
- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui
- OGL (WebGL)

## Getting Started
Prerequisites: Node.js + npm

```sh
npm install
npm run dev
```

## Scripts
```sh
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```
